# Spartan Quivers in Cape Slot

Small data-only NeoForge addon for Minecraft 1.21.1. It moves every item in
`#spartan_weaponry_unofficial:quivers` from the Accessories/Curios `back` slot
to `cape`.

The addon does not replace Spartan's Curios capability. Spartan scans all
equipped Curios for `QuiverBaseItem`, so ammo selection, firing, collection,
rendering and the quiver inventory remain handled by the original mod.

Installing the JAR on both client and server is required for MP so slot validation uses identical tags.
